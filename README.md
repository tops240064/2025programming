# 가계부 관리 시스템

Python Streamlit을 사용한 웹 기반 가계부 관리 애플리케이션입니다.

## 프로젝트 개요

이 프로젝트는 개인 또는 가족의 지출을 체계적으로 관리하고 분석할 수 있는 가계부 웹 애플리케이션입니다. 사용자가 입력한 데이터를 기반으로 통계, 그래프, AI 분석 기능을 제공합니다.

### 주요 기능

1. **가계부 입력 및 관리**
   - 날짜, 품목, 제품명, 가격, 개수 입력
   - 개당 가격 또는 전체 가격 입력 지원
   - 자동 가격 계산 (개당가격 ↔ 전체가격)
   - 필수 항목 검증 (날짜, 품목, 제품명, 가격)

2. **품목 추천 기능**
   - 제품명 키워드 기반 자동 품목 추천
   - 8가지 카테고리 지원: 식비, 교통비, 쇼핑, 생활비, 의료, 교육, 오락, 기타

3. **가계부 목록 및 필터링**
   - 날짜 범위 필터
   - 품목별 필터
   - 제품명 검색
   - 항목 삭제 기능

4. **통계 및 그래프**
   - 특정 기간 동안 특정 품목의 지출량 표시
   - 일별 지출 추이 그래프
   - 품목별 지출 비율 파이 차트
   - 품목별 지출 막대 그래프
   - 통계 요약 (총 지출, 평균, 최대, 거래 횟수)

5. **AI 분석 기능**
   - 사용자 쿼리 기반 데이터 분석
   - 품목별 지출 변화율 분석
   - 낱개 제품 구매 빈도 분석
   - 이전 기간과의 비교 분석

### 기술 스택

- **Python 3.7+**
- **Streamlit**: 웹 UI 프레임워크
- **Pandas**: 데이터 처리 및 분석
- **Plotly**: 인터랙티브 그래프 생성
- **JSON**: 데이터 저장 (파일 기반)

## 설치 및 실행 방법

### 1. 필수 패키지 설치

```bash
pip install -r requirements.txt
```

또는 개별 설치:

```bash
pip install streamlit pandas plotly
```

### 2. 애플리케이션 실행

#### 방법 1: 실행 스크립트 사용 (권장)

```bash
python streamlit_run_app.py
```

이 스크립트는 자동으로 필요한 패키지를 확인하고 설치한 후 애플리케이션을 실행합니다.

#### 방법 2: Streamlit 직접 실행

```bash
streamlit run app.py
```

### 3. 브라우저 접속

애플리케이션 실행 후 자동으로 브라우저가 열립니다. 
또는 수동으로 `http://localhost:8501`에 접속하세요.

## 사용 방법

### 가계부 입력

1. 좌측 사이드바에서 "가계부 입력" 메뉴 선택
2. 구매 날짜, 제품명, 품목, 가격, 개수 입력
3. 제품명 입력 시 자동으로 품목이 추천됩니다
4. 개당 가격 또는 전체 가격 중 하나를 선택하여 입력
5. "항목 추가" 버튼 클릭

### 가계부 목록 확인

1. "가계부 목록" 메뉴 선택
2. 날짜 필터, 품목 필터, 제품명 검색으로 원하는 항목 찾기
3. 삭제할 항목 선택 후 "선택한 항목 삭제" 버튼 클릭

### 통계 및 그래프 확인

1. "통계 및 그래프" 메뉴 선택
2. 분석할 기간과 품목 선택
3. 자동으로 표와 그래프가 생성됩니다

### AI 분석 사용

1. "AI 분석" 메뉴 선택
2. 분석할 기간 선택
3. 분석 요청 입력 (예: "식비 관련 지출과 낱개 제품 구매 빈도를 분석해줘")
4. "분석 실행" 버튼 클릭

## 데이터 저장

모든 가계부 데이터는 `household_data.json` 파일에 저장됩니다.
이 파일은 애플리케이션과 같은 디렉토리에 생성됩니다.

## 주의사항

- 필수 항목(날짜, 품목, 제품명, 가격)이 입력되지 않으면 항목 추가가 거부됩니다
- 데이터는 로컬 파일에 저장되므로 브라우저를 닫아도 유지됩니다
- 데이터 백업을 위해 `household_data.json` 파일을 정기적으로 백업하세요

## 프로젝트 구조

```

├── app.py                  # 메인 애플리케이션 코드
```
